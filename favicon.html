Tables

Each table is defined by a td, which stands for table data. Everything between td is a content of a table cell. A table cell can contain all sorts of elements including text, images, lists, links or even other tables.
To make some of the cells header cells use th instead of td.th stands for table header. By use of the border collapse, the two lined borders become one if you set it to border collapse
By use of border radius and setting the number of pixels, you get rounded borders.
In border radius, to skip the border around the overall table, leave out the table in the css selector.
