Image maps

The background image

The background image of a page is specified in the body section. If the background image cannot fit in the page, it repeats itself. To prevent it from repeating itself, set the background repeat to no repeat.

The picture element

The picture element allows you to display different pictures for different devices or screen sizes It contains more than one image each refering to a different screen size.this way the browser chooses according to the screen size. It can also be used to set images of different formats. The browser uses the first image format it recognises.
